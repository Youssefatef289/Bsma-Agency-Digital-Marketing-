# دليل البدء السريع

## خطوات التشغيل

1. **تثبيت المكتبات:**
   ```bash
   npm install
   ```

2. **تشغيل المشروع:**
   ```bash
   npm run dev
   ```

3. **فتح المتصفح:**
   - سيتم فتح المتصفح تلقائياً على `http://localhost:3000`
   - أو افتح المتصفح يدوياً على العنوان أعلاه

## ملاحظات مهمة

### تحديث معلومات التواصل
قبل النشر، قم بتحديث:
- رقم الهاتف في `src/components/Footer.jsx` و `src/pages/Contact.jsx`
- البريد الإلكتروني في نفس الملفات
- رقم واتساب في `src/pages/Contact.jsx` (المتغير `whatsappNumber`)
- روابط السوشيال ميديا في `src/components/Footer.jsx` و `src/pages/Contact.jsx`

### بناء المشروع للنشر
```bash
npm run build
```
الملفات الجاهزة ستكون في مجلد `dist`

## الصفحات المتاحة

- `/` - الصفحة الرئيسية
- `/about` - من نحن
- `/contact` - تواصل معنا

## المتطلبات

- Node.js 16+ 
- npm أو yarn
