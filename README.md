# بصمة للتسويق الرقمي - Bama Agency Digital Marketing

موقع تعريفي احترافي لشركة بصمة للتسويق الرقمي، مبني باستخدام React و Tailwind CSS.

## المميزات

- ✅ تصميم عصري ومتجاوب 100% على جميع الأجهزة
- ✅ صفحات متعددة: الرئيسية، من نحن، تواصل معنا
- ✅ عرض احترافي للخدمات في كروت تفاعلية
- ✅ فورم تواصل كامل مع زر واتساب
- ✅ Animations خفيفة وسلسة
- ✅ دعم كامل للغة العربية (RTL)
- ✅ كود منظم وقابل لإعادة الاستخدام

## التقنيات المستخدمة

- **React 18** - مكتبة JavaScript لبناء واجهات المستخدم
- **Vite** - أداة بناء سريعة
- **Tailwind CSS** - إطار عمل CSS للتصميم
- **React Router** - للتنقل بين الصفحات
- **Lucide React** - أيقونات احترافية

## التثبيت والتشغيل

### المتطلبات
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

### خطوات التثبيت

1. تثبيت المكتبات:
```bash
npm install
```

2. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

3. بناء المشروع للإنتاج:
```bash
npm run build
```

4. معاينة النسخة المبنية:
```bash
npm run preview
```

## الصفحات

### الرئيسية
- Hero Section مع عنوان رئيسي
- عرض المميزات
- عرض الخدمات في كروت احترافية
- قسم CTA للتواصل

### من نحن
- قصة الشركة
- الرؤية والرسالة
- الإحصائيات
- لماذا تختار بصمة

### تواصل معنا
- فورم تواصل كامل (اسم، هاتف، إيميل، رسالة)
- معلومات التواصل
- زر واتساب مباشر
- روابط السوشيال ميديا

## التخصيص

### تغيير معلومات التواصل
قم بتعديل المعلومات في:
- `src/components/Footer.jsx`
- `src/pages/Contact.jsx`

### تغيير رقم واتساب
في `src/pages/Contact.jsx`، قم بتعديل المتغير `whatsappNumber`.

### إضافة صور
ضع الصور في مجلد `public/image/` وأعد استخدامها في المكونات.

## النشر

المشروع جاهز للنشر على:
- Vercel
- Netlify
- GitHub Pages
- أي خدمة استضافة تدعم React

```bash
npm run build
```

ثم ارفع محتويات مجلد `dist` إلى الخادم.
